# ASTGCN 地铁全线路人流量预测

## 概述
基于 ASTGCN（Attention Spatial-Temporal Graph Convolutional Network）对杭州地铁 A/B/C 三线共 81 站进行人流量预测。

## 数据范围
- **线路**: A(14站) + B(34站) + C(32站)，共 80 站（54号缺失）
- **时段**: 2019-01-01 ~ 2019-01-25，25 天
- **粒度**: 10 分钟，每天 144 个时段
- **特征**: 时间特征 + 进站/出站人数

## 模型架构
```
输入 (B, T_h=12, N=81, F=7)
  → 线性投影 → (B, 12, 81, C)
  → ST Block × N（时间注意力 + 空间注意力 + GCN + 时间卷积）
  → 输出层 → (B, T_p, N, 2)
```

## 数据划分

```
训练集: 2019-01-01 ~ 2019-01-22 (22天) → 3,156 个滑动窗口
验证集: 2019-01-23 ~ 2019-01-25 (3天)  →   427 个滑动窗口
```

拆分依据：预测目标第一个时段所属日期 ≤ 01-22 为训练，否则为验证。

---

## 预测方式：多步直接预测

```
每个窗口独立预测，输入全部使用真实历史，不使用上一个窗口的预测值：

窗口1: [t0  ... t11]  → 预测 [t12 ... t17]    ← 输入全为真实值
窗口2: [t1  ... t12]  → 预测 [t13 ... t18]    ← 输入全为真实值
窗口3: [t2  ... t13]  → 预测 [t14 ... t19]    ← 输入全为真实值
```

**优点**：误差不累积，每个窗口都从真实数据出发  
**局限**：部署时如需预测 > 60min，需改用自回归或扩大 T_pred

---

## 窗口大小选择

当前 `T_hist=12`（2小时）、`T_pred=6`（1小时）为经验选择，
参考了交通预测文献常用设置。未使用 ACF/PACF 做严格时序自相关分析，
后续可补充。
